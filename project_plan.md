# بصيرة 4×4 - خطة المشروع

## 1. وصف المشروع
موقع رسمي لبرنامج **بصيرة 4×4** المقدم من **وايت روك القابضة**، برنامج معرفي تحفيزي يستهدف تمكين الأفراد من التحول الذاتي الواعي عبر أربعة كتب عالمية مؤثرة. الموقع يستهدف الأفراد الطموحين عالمياً ويتيح التسجيل وتقديم الاستفسارات.

## 2. هيكل الصفحات
- `/` - الصفحة الرئيسية (Hero, مقدمة, الكتب الأربعة, الجمهور, التسجيل, اتصل بنا)
- `/about` - عن البرنامج (من نحن، الرؤية والرسالة، الأهداف، نقاط القوة)
- `/structure` - هيكلية البرنامج (الكتب الأربعة بالتفصيل)

## 3. المميزات الأساسية
- [x] صفحة رئيسية بتصميم احترافي RTL عربي
- [x] صفحة عن البرنامج
- [x] صفحة هيكلية البرنامج (الكتب الأربعة)
- [x] نموذج تسجيل (Readdy Forms)
- [x] نموذج التواصل (Readdy Forms)
- [x] تنقل ثابت شفاف يتحول لأبيض عند التمرير
- [ ] قسم الجمهور المستهدف مع خريطة توسع عالمي

## 4. تصميم البيانات
لا حاجة لقاعدة بيانات - النماذج تُرسل عبر Readdy Forms.

## 5. خطة التكامل مع الأطراف الثالثة
- Supabase: غير مطلوب
- Shopify: غير مطلوب
- Stripe: غير مطلوب
- Readdy Forms: لنموذج التسجيل ونموذج الاستفسار
